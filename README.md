# Fixground Landing Page

Static bilingual landing page for Fixground.

## Structure
- `index.html`: redirects to `./kr/`
- `kr/index.html`: Korean page
- `en/index.html`: English page
- `assets/styles.css`: shared styles
- `assets/main.js`: shared interactions (hero stagger, section reveal, sticky nav effect)

## Deploy to GitHub Pages
1. Push this repository to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your default branch and `/ (root)` folder.
5. Save and wait for deploy.

Published routes:
- Korean: `/kr/`
- English: `/en/`
